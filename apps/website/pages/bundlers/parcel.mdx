# Usage with Parcel

### Installation

```sh
# npm
npm i -D @wyw-in-js/parcel-transformer
# yarn
yarn add --dev @wyw-in-js/parcel-transformer
# pnpm
pnpm add -D @wyw-in-js/parcel-transformer
# bun
bun add -d @wyw-in-js/parcel-transformer
```

> Note: WyW packages are ESM-only and require Node.js `>=22`.

### Configuration

Prepend `@wyw-in-js/parcel-transformer` to the default JS pipeline in `.parcelrc`:

```json
{
  "extends": "@parcel/config-default",
  "transformers": {
    "*.{js,mjs,jsm,jsx,es6,cjs,ts,tsx}": ["@wyw-in-js/parcel-transformer", "..."]
  }
}
```

`eval.require` defaults to `warn-and-run` in v2. If eval-time `require()` happens, WyW will warn and execute it at
runtime; use `importOverrides` or `eval.require: 'error'` to make it explicit.
