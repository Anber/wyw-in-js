# `@wyw-in-js/cli`

Bundler-agnostic extraction CLI for WyW-in-JS.

## Install

```sh
# npm
npm i -D @wyw-in-js/cli
# yarn
yarn add -D @wyw-in-js/cli
# pnpm
pnpm add -D @wyw-in-js/cli
# bun
bun add -d @wyw-in-js/cli
```

## Usage

```sh
wyw-in-js --help
```

Docs and examples: https://wyw-in-js.dev/cli
